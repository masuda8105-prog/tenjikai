# 商品データの自動読み込み

このフォルダには、GitHub Pages で静的配信できる商品マスタを配置しています。

## 配置ファイル

- `index.html`: 起動時に商品マスタを自動読み込みする改修済みWebアプリ
- `products.json`: Web アプリの起動時に自動読み込みする商品データ
- `products.csv`: 元CSVをUTF-8に変換した商品データ

現在の商品件数: 3,861件

## 商品データの更新方法

1. 最新の商品CSVを用意します。
2. CSVの列を次の形式にします。
   - `No.`
   - `商品名`
   - `卸価格`
3. `products.csv` を差し替えます。
4. `products.json` も同じ内容で更新します。
   - JSON形式は `{ "code": "品番", "name": "商品名", "price": 価格 }` の配列です。
5. GitHub Pages に公開している場合は、変更をコミットしてpushします。

## 既存Webアプリへの組み込み方針

ページ起動時に `products.json` を `fetch()` で読み込み、成功したら既存の商品配列に反映します。
失敗した場合は従来のExcel/CSV手動読み込み機能をそのまま使える状態にします。

表示文言の例:

- 自動読込済み：商品データ 3,861件
- 自動読込に失敗しました。設定からExcelまたはCSVを読み込んでください。
- 手動読込済み：商品データ 3,861件

GitHub Pages ではアプリと同じリポジトリ内の静的ファイルとして `products.json` を置けば、相対パス `./products.json` で読み込めます。
